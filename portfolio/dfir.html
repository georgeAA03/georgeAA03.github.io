Project Overview

The DFIR project focuses on investigating compromised systems, analyzing logs and memory, and performing forensic analysis to simulate real-world enterprise incident response. It emphasizes evidence preservation, threat detection, and remediation planning in an ongoing SOC environment.

  • Tools: Autopsy, Volatility, FTK Imager, Wireshark, Sysinternals Suite
  • Techniques: Disk & memory analysis, log correlation, network traffic investigation, artifact recovery
  • Outcomes: Pending (project ongoing)
  • Deliverable: Final report and analysis (project ongoing)

Realistic Enterprise Scenario

Simulates a real-world SOC incident response workflow with compromised endpoints, anomalous network traffic, and suspicious logs. Focus is on threat detection, containment, remediation, and clear documentation for each stage of the investigation.

Techniques and Skills to be Demonstrated

  • Memory analysis and forensic imaging
  • Disk image examination and artifact recovery
  • Network traffic analysis and log correlation
  • Identification of malware, unauthorized access, and user activity
  • Documentation of findings and recommended security measures
