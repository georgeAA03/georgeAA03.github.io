Project overview

Ethical Hacking 1 is an end-to-end penetration test focused on a vulnerable Linux target. The engagement covers reconnaissance, service and web enumeration, exploitation attempts, and privilege escalation analysis. The deliverable is a detailed LaTeX walkthrough containing commands, outputs, and remediation recommendations.

  • Tools: Nmap, Nexpose, Metasploit, Enum4Linux, LinPEAS, Burp Suite
  • Techniques: Recon, web-app enumeration (ports 8000/8080), SUID analysis, PATH abuse, local privilege escalation
  • Outcomes: User & root flags obtained, credential extraction, documented exploitation paths and PoCs
  • Deliverable: Full LaTeX walkthrough with step-by-step commands, screenshots, and prioritized remediation

Realistic Enterprise Scenario

This lab was structured to simulate a real-world security audit. Each attack vector (on ports 22, 8000, and 8080) represented distinct layers of system exposure, illustrating how vulnerabilities can chain together to achieve privilege escalation and full system compromise. The scenario emphasizes systematic enumeration and the importance of prioritized remediation.

Techniques and Skills Demonstrated

  • Network reconnaissance and service scanning to identify exposed services and versions
  • Web application and metadata discovery to identify sensitive files and attack vectors
  • SUID and filesystem analysis to identify local escalation opportunities
  • Exploit validation and post-exploitation enumeration to capture proof-of-concept artifacts
