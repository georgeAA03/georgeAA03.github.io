Project Overview

This Final Year Project proposes an AI-enhanced ELK architecture integrating real-time threat detection, adaptive learning, and automated incident response in a scalable and transparent framework. Logs from Windows systems (Sysmon, Winlogbeat), network captures (Packetbeat, Arkime), and honeypots (T-Pot) are collected and enriched via Logstash, then indexed in Elasticsearch. Unsupervised ML algorithms (Isolation Forest, K-Means) detect anomalies, while supervised models (Random Forest, CNNs) classify security events. Adaptive learning retrains models using analyst feedback to adapt to emerging threats. Kibana dashboards visualize alerts and trends in real time, and the system can automatically trigger responses such as alerts, IP blocking, or MISP incident reporting. This hybrid approach enhances visibility via ELK dashboards and intelligence via AI-driven analysis.

  • ELK Stack (Elasticsearch, Logstash, Kibana)
  • Windows Log Collection (Sysmon + Winlogbeat)
  • AI/ML Threat Detection
  • Network Capture & Analysis (Arkime, Packetbeat)
  • Threat Intelligence (VirusTotal API)
  • Honeypot Telemetry (T-Pot)
  • Real-Time Dashboards (Kibana)

Realistic Enterprise Scenario

The project simulates a real-world Security Operations Center (SOC) environment, aggregating host and network logs, honeypot data, and threat intelligence into a centralized SIEM. Integration of Arkime and VirusTotal supports deep network analysis and automated enrichment. Machine learning models such as Isolation Forest and Autoencoders detect suspicious behavior that traditional detection mechanisms might miss.

Techniques and Skills to be Demonstrated
