Project Overview

This project showcases a comprehensive penetration test on both Linux and Windows/Active Directory environments. I performed structured reconnaissance, service and web application enumeration, exploitation, and post-exploitation analysis, demonstrating how vulnerabilities can be chained to achieve elevated access. The full walkthrough includes commands, outputs, screenshots, and recommendations for system hardening.

  • Tools: Nmap, Nexpose, Metasploit, LinPEAS, CrackMapExec, BloodHound, Impacket, Burp Suite
  • Techniques: Recon, LFI/metadata extraction, SSH brute-force, SMB & AD enumeration, Kerberoasting, Golden/Silver tickets
  • Outcomes: Initial access, privilege escalation, credential recovery, domain artifacts captured
  • Deliverable: Full LaTeX walkthrough with evidence, screenshots, and remediation plan

Realistic Enterprise Scenario

This lab was structured to simulate a real-world security audit. Each attack vector represented distinct layers of system exposure, illustrating how vulnerabilities can chain together to achieve privilege escalation and full system compromise. The scenario provided practical insights into multi-layered defenses, enumeration strategies, and the importance of prioritized remediation.

Techniques and Skills Demonstrated

  • Automated and manual reconnaissance for comprehensive attack surface mapping
  • Web application and metadata discovery to identify LFI and sensitive file exposure
  • Windows Active Directory enumeration and exploitation techniques
  • Post-exploitation and privilege escalation using LinPEAS and manual enumeration
  • Clear documentation with remediation recommendations for improving security posture
